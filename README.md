# Projet IPS Prod

## Compilation
Pour compiler :
> `make`

Pour compiler les tests :
> `make tests`

Pour générer la documentation :
> `make doc`

Pour nettoyer les fichiers compilés :
> `make clean`

## Execution
Pour tout lancer et générer les figures :
> `./run.sh`

## Dépendances
- armadillo >= 9.900
- python3 avec matplotlib et numpy
- povray >= 3.7.0.8
